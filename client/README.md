## 📖 About the Project

**Kifahari Styles** is a professional website built for a real local business in Nairobi, Kenya. The business offers three categories of services — a Ladies Salon, a Gentlemen's Barbershop and a Fashion Boutique — all under one roof on Ngong Road, Nairobi.

*Kifahari* is a Swahili word meaning **Luxurious** — and the website reflects exactly that with a gold and black luxury design theme.

This project was built as part of the **Future Interns Full Stack Web Development Internship (Task 3)** — Build, Pitch & Monetize a Real Local Business Website.

---

## 🎯 Objective

Build a professional website for a local business that helps it:
- Look more professional online
- Attract new customers
- Increase trust and credibility
- Grow its online presence

---

## ✨ Features

- **Home Page** — Hero section, services preview, about strip, gallery preview, testimonials and CTA
- **Services Page** — Full categorized menu with pricing, HOT badges and duration for Salon, Barbershop and Boutique
- **About Page** — Brand story, company timeline, values and team section
- **Gallery Page** — 36 real photos with category filters, uniform grid and lightbox navigation
- **Contact Page** — Booking form with Formspree integration, location info, WhatsApp button and Google Maps
- **Responsive Design** — Fully mobile-friendly on all screen sizes
- **Luxury Theme** — Gold and black color scheme throughout
- **Smooth Animations** — Scroll reveal, hover effects and transitions

## 🛠️ Tech Stack

### Frontend
- **React.js** — UI framework
- **Vite** — Build tool and dev server
- **React Router DOM** — Client-side routing
- **CSS-in-JS** — Inline styles with responsive media queries

### Form Handling
- **Formspree** — Contact form email delivery (no backend needed)

### Deployment
- **Netlify** — Frontend hosting

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- Git installed

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/JANE-dot978/FUTURE_FS_03.git
cd FUTURE_FS_03
```

**2. Install dependencies**
```bash
cd client
npm install
```

**3. Start development server**
```bash
npm run dev
```

**4. Open in browser**
```
http://localhost:5173
```

### Build for Production
```bash
npm run build
```

## 📢 Pitch Explanation

### Who is the Business?
Kifahari Styles is a multi-service style destination in Nairobi offering a Ladies Salon, Gentlemen's Barbershop and Fashion Boutique all under one roof.

### What Problem Does the Website Solve?
Before this website, Kifahari Styles had no online presence. Potential customers had no way to:
- Find the business online
- View services and prices before visiting
- Book appointments in advance
- See photos of the work done

### How Does It Help Attract Customers?
1. **Visibility** — The business now appears online and can be found by anyone searching for salons or barbershops in Nairobi
2. **Trust** — Professional design builds credibility and makes the business look established
3. **Convenience** — Customers can view all services and prices, then book via WhatsApp or the contact form
4. **Gallery** — Real photos of work done showcase quality and attract new clients
5. **Mobile Friendly** — Most Nairobi customers browse on phones — the site works perfectly on all devices
